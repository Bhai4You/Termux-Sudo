# Termux-Sudo
Lazy Geeks Script For Sudo


#Installation..

1) apt install git


2) git clone https://github.com/Bhai4You/Termux-Sudo


3) cd Termux-Sudo


4) chmod +x sudo.sh


5) bash sudo.sh

Note : This will not work unless you have
Superuser access . I have got the pipeline for a sudo script
That works without root.
